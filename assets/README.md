# Assets Folder

Folder ini berisi semua aset yang digunakan dalam website Wisata Daerahku.

## Struktur Folder

- **css/**: Berisi file stylesheet custom (style.css)
- **js/**: Berisi file JavaScript untuk interaktivitas (script.js)
- **img/**: Folder untuk menyimpan gambar-gambar destinasi wisata

## Catatan

Pastikan semua file CSS dan JS sudah terhubung dengan benar di setiap halaman HTML.
