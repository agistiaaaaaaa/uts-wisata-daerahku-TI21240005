# Wisata Daerahku

Website statis untuk memperkenalkan destinasi wisata di Lombok, lengkap dengan informasi beranda, daftar destinasi, galeri foto, dan halaman kontak.

## Fitur Utama
- Navigasi antar halaman (beranda, destinasi, galeri, kontak).
- Hero banner dengan ajakan untuk menjelajah destinasi.
- Highlight kategori wisata: alam, budaya, kuliner.
- Halaman detail destinasi dengan informasi lebih lengkap.
- Galeri foto interaktif dan formulir kontak sederhana.

## Teknologi yang Digunakan
- HTML5 & CSS3
- Bootstrap 5 dan Bootstrap Icons via CDN
- Tailwind CSS CDN untuk utilitas tambahan
- JavaScript untuk interaksi dasar

## Struktur Proyek
```
wisata-daerahku/
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── img/
│   │   └── ... (gambar destinasi)
│   └── js/
│       └── script.js
├── destinasi.html
├── detail.html
├── galeri.html
├── index.html
└── kontak.html
```

## Cara Menjalankan
1. Clone repositori ini atau unduh arsipnya.
2. Buka file `index.html` langsung di browser.
3. Gunakan navigasi untuk menjelajahi halaman lainnya.
